Color Scheme

Primary color: #4b4b4d

This color will be used for all texts and titles in the main page. It will also be used as the background color for the header and footer.

Secondary color: #0098da

This orange color is present in the logo, and also in the navigation bar.

Accent 1: #f58634

This color will be used in the logo, in the navigation bar when the a element is hoover, and also when the a element is with the active class. It will also be used in some text in the footer.

Accent 2: #ffffff

The white color will be used in texts that are in the footer, and also in the text of the a elements of the navigation bar.
